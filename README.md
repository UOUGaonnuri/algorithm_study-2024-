# 알고리즘 스터디 (심화편)
## 멤버
- 신재희 (진행자)
- 최정호
- 박서준
- 진명재
- 김현수
## 진행 방법
- 시간 : 매주 월요일 오후 10시
- 알고리즘 강의
- 해당 알고리즘 관련 문제
- 코드 리뷰
    - 깃 허브에 코드 제출 및 PR을 통한 코드리뷰
## 깃 허브
- 브랜치 : 각 멤버의 아이디
    - 예 : jaljayo85
- 커밋 메세지 : [N주차] ({푼 문제 수} / {이번 주차 문제 수})
    - 예 : [10주차] (3 / 5)
- 문제 제출 : 자신의 브랜치에 해당 문제 번호로 작성한 문제 번호로 제출
    - 예 : 1717.java or 1717.cpp
- PR 제목 : [N주차] 과제 제출합니다.
    - 예 : [10주차] 과제 제출합니다.
- 최종 PR 메세지 : 문제 난이도 리뷰 바랍니다.
